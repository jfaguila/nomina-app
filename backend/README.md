# Backend API para Nómina App

## API Endpoints
- `POST /api/verify-nomina` - Procesar nómina con OCR
- `POST /api/validate-data` - Validar datos corregidos
- `GET /` - Status del servidor

## Deploy en Railway
Este backend está configurado para desplegarse en Railway como servicio independiente.

## Variables de Entorno
- `PORT` - Puerto del servidor (por defecto: 5987)
